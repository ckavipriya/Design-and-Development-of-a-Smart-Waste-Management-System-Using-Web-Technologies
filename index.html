Mission Statement

EcoSense aims to revolutionize urban sanitation by bridging the gap between physical waste infrastructure and digital management systems. By providing real-time visibility into waste generation, we enable authorities to act proactively rather than reactively.

How It Works

The workflow from sensor to solution.

1

Data Collection

IoT sensors inside bins measure waste levels and transmit data via GSM.

2

Cloud Processing

The Node.js backend receives data, validates it, and stores it in MongoDB.

3

Analytics & Action

Admins view the dashboard. The system suggests optimal truck routes.
