Smart Waste Management System

This project focuses on developing a web-based system that helps monitor waste levels in bins across different areas.

The system uses web technologies such as HTML, CSS and JavaScript to simulate a smart waste monitoring platform.

Main objectives:

  • Monitor garbage levels
  • Send alerts when bins are full
  • Improve waste collection efficiency
  • Maintain clean and sustainable cities
